安装Docker

使用方式

build一个新的镜像

1.$ git clone https://github.com/wangmax0330/docker-jdk-tomcat.git
2.$ cd docker-jdk-tomcat
3.$ sudo docker build -t docker-jdk-tomcat .

运行镜像

1.$ sudo docker run -d -p 8090:8080 docker-jdk-tomcat

打开http://<host>:8090可以看到tomcat 首页

常用命令

  • 进去运行时的容器
1.$ docker exec -it <contain id> /bin/bash
  • 列出当前所有正在运行的container 
1.$docker ps
  • 列出所有的container (包括历史)
1.$docker ps -a
  • 列出最近一次启动的container 
1.$docker ps -l

Maven 远程部署Tomcat 项目

pom.xml配置部署的server

1.<plugin>
2.    <groupId>org.apache.tomcat.maven</groupId>
3.    <artifactId>tomcat7-maven-plugin</artifactId>
4.    <version>2.2</version>
5.    <configuration>
6.        <url>http://121.42.212.240:8090/manager/text</url>
7.        <server>TomcatServer</server>
8.        <path>/sample</path>
9.        <username>methew</username>
10.        <password>methewAdmin</password>
11.    </configuration>
12.</plugin>
